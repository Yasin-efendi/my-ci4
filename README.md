# Blog dengan CodeIgniter 4 & Tailwind CSS

Ini adalah project web blog sederhana menggunakan CodeIgniter 4 dan Tailwind CSS.

## Fitur yang akan dikembangkan:

- Authentication
- Admin Dashboard
- CRUD Posts & Categories
- Frontend Blog
- Responsive Design

## Cara Menjalankan:

1. Clone repository ini
2. Jalankan `composer install`
3. Setup database di file `.env`
4. Jalankan migration dan seeder
